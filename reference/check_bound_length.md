# Check performed within the [`get_bounds`](https://github.com/ndpvh/discounting-affect/reference/get_bounds.md) function

Check performed within the
[`get_bounds`](https://github.com/ndpvh/discounting-affect/reference/get_bounds.md)
function

## Usage

``` r
check_bound_length(n, lower, upper)
```

## Arguments

- n:

  Integer denoting the needed length of the bounds

- lower, upper:

  Numeric vector containing the user-specified lower and upper bounds
  for which the length should be checked.

## Value

List containing the corrected lower and upper bounds (in that order)
